# 贡献指南

感谢你对 hello天气 项目的关注！

## 如何贡献

### 1. 提交问题
- 发现Bug？请提交 [Issue](https://github.com/HenryMax2025/hello-weather/issues)
- 有新功能建议？请提交 [Feature Request](https://github.com/HenryMax2025/hello-weather/issues)

### 2. 提交代码
1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送分支 (`git push origin feature/AmazingFeature`)
5. 创建一个 Pull Request

### 3. 代码规范
- 使用 Kotlin 官方代码风格
- 遵循 Android 开发最佳实践
- 确保代码通过现有的测试

## 开发环境要求

- Android Studio Arctic Fox 或更高版本
- JDK 17+
- Gradle 9.4+

## 本地构建

```bash
# 克隆项目
git clone https://github.com/HenryMax2025/hello-weather.git

# 进入目录
cd hello-weather

# 编译 Debug 版本
./gradlew assembleDebug
```
